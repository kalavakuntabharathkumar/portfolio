## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This project is configured for GitHub Pages deployment. The build output is in `dist/` and the base URL is set to `/portfolio/`.

To deploy manually:
```bash
npm run build
# Then push the dist/ folder to the gh-pages branch
```
